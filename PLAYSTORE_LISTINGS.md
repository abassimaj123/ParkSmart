## Store Listing Guidelines

### Screenshots (8 per app, 1080x1920 minimum)
For each app, capture:
1. **Calculator Screen**: Main input fields with example data
2. **Results Screen**: Monthly payment + total interest prominently displayed
3. **Chart/Graph**: Visual breakdown (amortization or comparison)
4. **History Screen**: Sample saved calculations
5. **PDF Export**: Share dialog showing PDF option
6. **Premium Paywall**: "Unlock Premium" call-to-action
7. **Settings Screen**: Language/theme toggle
8. **Dark Mode**: Screenshot of same screen in dark theme

### Feature Graphics (1024x500 px)
Design with:
- App name + tagline (e.g., "Mortgage Calculator — Find Your Best Rate")
- Key benefit call-out (e.g., "Compare 4 Offers Instantly")
- Brand colors (navy #1B3A6B primary, accent color per app)
- Clean typography, no cluttered design

### Video Preview (Optional, 15–30 sec)
1. App launch animation
2. Calculator demo (input values)
3. Results display (highlight key output)
4. Chart/comparison visualization
5. Paywall showcase
6. Text overlay: "Download [App] on Google Play"
